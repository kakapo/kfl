  KFL是一个轻快的、友好的、MVC模式的PHP开发框架，基于应用层解决Webserver集群带来的系列问题。

  • 统一配置，一键同步。
  • 支持数据片和网页的分布式缓存。
  • 支持Session会话数据同步和在线用户统计。
  • 支持错误日志和应用服务监控。
  • 集成用户数据文件的分布式存储服务。
