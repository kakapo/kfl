数据库的缓存配置

  • 1、支持本地file，多级目录缓存支持
  • 2、分布式内存缓存

    • 影响到方法：getRow(),getOne(),getAll(); 使用缓存举例：
    1、简单：
    $db->getAll("select * from table ",'',3600);

    2、指定cacheId：
    $db->getRow("select * from table where id=?",array(5),3600,'table5');

    不使用缓存举例：
    1、$db->getRow("select * from table where id=?",array(5));

    位置: 缓存管理 > 数据片缓存设置
    变量 值 备注
    cacheOpen 启动缓存开关,关闭=0，开启=1
    cacheStore 缓存存储方式,支持(file|memcache)
    cacheTime 默认缓存时间，秒
    cacheDir 缓存本地路径，cacheStore=file时有效
    cacheServer Memcached服务器，cacheStore=memcache时有效
