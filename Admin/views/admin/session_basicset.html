数据库的缓存配置

  • 1、支持本地file，多级目录缓存支持
  • 2、分布式内存缓存

    • 影响到方法：getRow(),getOne(),getAll(); 使用缓存举例：
    1、简单：
    $db->getAll("select * from table ",'',3600);

    2、指定cacheId：
    $db->getRow("select * from table where id=?",array(5),3600,'table5');

    不使用缓存举例：
    1、$db->getRow("select * from table where id=?",array(5));

    位置: 会话管理 > 基本设置
    变量 值 备注
    sessionHandle Session 存储方式
    lifeTime 默认缓存时间，秒
    dataBase 数据库名称，sessionHandle=database时有效
    memcached Memcached服务器，sessionHandle=memcache时有效
