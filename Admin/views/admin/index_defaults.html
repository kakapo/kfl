  KFL是之前KFC框架的轻量级版本，框架KFL目录下所有文件大小35k，主要去除了KFC的组件概念，保留了MVC的开发模式，特别适合前台的网站开发。

  • 使用MVC开发模式。
  • 引入文件少，启动快速。
  • 具备针对访问路径的页面自动缓存功能；
  • 模板本身就是使用php语法，简单友好；
  • 采用pdo进行数据库访问，高速易用；
  • session 支持 file,mysql,memcache存储功能；
  • 单一入口访问网站，安全。
